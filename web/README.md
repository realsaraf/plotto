# Toatre Web

Next.js 16 web app for Toatre.

## Local development

From this folder, run:

```bash
npm run dev
```

The app serves on `http://localhost:3000` by default.

## Production build

```bash
npm run build
npm run start
```

## Deployment

This app deploys to DigitalOcean App Platform. The deployment spec lives in the repo root at `.do/app.yaml`.
